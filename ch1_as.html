1.11 Describe the difference between client-side programming and server-side programming.

Client-side programming takes place in the browser; server-side programming, on the other hand, takes place in the web server. Client-side programming has its limitations, due to security reasons; server-side programming can mirror data from the client and is thusly more secure.
